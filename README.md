# Shopping Cart

> This code was built following a tutorial: Redux Toolkit Tutorial ‚Äì JavaScript State Management Library. We can increment and decrease the quantities of the items in the cart, clear the cart, check the quantity of items in it and the total price.

![screenshot](./mobile-store.gif)

## Setup

### Clone this repository

```bash
$ git clone git@github.com:Sahar-AbdelSamad/shopping-cart.git
$ cd shopping-cart
```

### Run project

```bash
$ npm install
$ npm run start
```

### Run tests

To run the test just use the following command!

```bash
$ npm test
```

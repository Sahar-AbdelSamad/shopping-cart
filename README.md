# Shopping Cart

> This code was built following a [Redux Toolkit Tutorial ‚Äì JavaScript State Management Library](https://www.youtube.com/watch?v=bbkBuqC1rU4). It is a mobile store cart where we can increment and decrease the quantities of the items in the cart, clear the cart, check the quantity of items in it and the total price.

![screenshot](./mobile-store.gif)

## Setup

### Clone this repository

```bash
$ git clone git@github.com:Sahar-AbdelSamad/shopping-cart.git
$ cd shopping-cart
```

### Run project

```bash
$ npm install
$ npm run start
```
